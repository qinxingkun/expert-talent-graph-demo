# 专家人才关系子系统原型 (Expert Talent Graph Demo)

这是一个基于原生 HTML/CSS/JS 开发的**亿级科技知识图谱算法引擎 - 专家人才关系子系统**的原型演示页面。

## 功能特性

- **科技专家/人才直接关系展示**：展示算法支撑的业务板块和服务对象。
- **在线测试功能**：模拟专家数据的关系识别过程。
- **算法参数配置**：运维人员可实时调整算法权重参数。
- **API 接口文档**：为开发人员提供多语言 API 调用示例。
- **配置中心**：统一管理算法版本和采集规则。

## 部署说明

本项目支持直接部署至 GitHub Pages：

1. 将代码推送到 GitHub 仓库的 `main` 分支。
2. 在 GitHub 仓库设置中开启 GitHub Pages。
3. 选择 `main` 分支作为部署源。

## 运行环境

- 无需后端服务，直接打开 `index.html` 即可运行。
- 使用 Font Awesome 提供图标支持（需要联网）。
